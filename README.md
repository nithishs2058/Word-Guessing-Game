# 🎯 Word Guessing Game (Hangman) – Python Project

A simple Python-based word guessing game where the player must guess a hidden word letter by letter within a limited number of attempts. The game provides an interactive command-line experience and tests the player’s vocabulary and guessing skills.

---

## 📌 About The Project

This project is designed as a beginner-friendly Python application to demonstrate fundamental programming concepts through a fun game.

It covers concepts such as:

* Random word selection
* Loops and conditional statements
* Lists for tracking progress
* User input handling
* Basic game logic implementation

The game randomly selects a word, and the player must guess it before running out of attempts.

---

## 🎯 Key Functionalities

* Randomly selects a word from a predefined list
* Displays blanks for unguessed letters
* Updates correct guesses in real-time
* Limits the number of wrong attempts
* Displays win or lose result

## ▶️ How to Run the Program

1. Ensure Python is installed on your system
2. Save the file as `word_guess.py`
3. Open terminal or command prompt
4. Run the program using:

```bash id="run123"
python word_guess.py
```

---

## 🎮 How to Play

1. The program selects a random word
2. Blanks (`_`) represent each letter
3. Enter one letter at a time
4. Correct guesses reveal letters
5. Wrong guesses reduce attempts
6. Guess the word before attempts run out
